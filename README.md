# 剪贴板管理器

一个基于 Go 和 JavaScript 开发的单页面剪贴板管理应用。

## 功能特性

- 📌 **粘贴功能**：点击粘贴按钮，将系统剪贴板内容添加到列表
- 📋 **历史记录**：以列表形式展示所有粘贴的内容（最新的在最上面）
- 📑 **复制功能**：每个列表项都有复制按钮，可将内容复制回系统剪贴板
- 🗑️ **删除功能**：删除不需要的项目，删除前有确认提示
- 📍 **置顶功能**：重要内容可以置顶，置顶项目会显示在列表最上方
- 📄 **智能折叠**：超过 1000 字符的内容自动折叠，点击展开/收起
- 🔄 **自动刷新**：可开启自动刷新功能，每 2 秒自动更新列表
- 🎨 **美观界面**：渐变背景、动画效果、响应式设计

## 技术栈

- **后端**：Go 语言
  - 标准库 `net/http` 构建 HTTP 服务器
  - RESTful API 设计
  - 线程安全的数据管理

- **前端**：原生 JavaScript
  - Clipboard API 实现剪贴板读写
  - Fetch API 与后端通信
  - CSS3 动画和渐变效果

## 运行方法

### 1. 确保已安装 Go

```bash
go version
```

### 2. 运行应用

```bash
cd clipboard-app
go run main.go
```

### 3. 访问应用

在浏览器中打开：`http://localhost:8084`

## 使用说明

1. **粘贴内容**
   - 先在系统中复制一些文本（Ctrl+C 或 Cmd+C）
   - 点击页面上的"📌 粘贴剪贴板内容"按钮
   - 内容会出现在列表最上方

2. **复制内容**
   - 点击列表项右侧的"复制"按钮
   - 内容会被复制到系统剪贴板
   - 可以在其他应用中粘贴使用（Ctrl+V 或 Cmd+V）

3. **置顶内容**
   - 点击"置顶"按钮将重要内容固定在列表顶部
   - 置顶的项目会显示黄色背景和 📌 标记
   - 再次点击可取消置顶

4. **删除内容**
   - 点击"删除"按钮
   - 会弹出确认对话框，确认后删除

5. **查看长文本**
   - 超过 1000 字符的内容会自动折叠
   - 点击内容区域展开查看全部
   - 再次点击收起

6. **自动刷新**
   - 点击右上角的开关启用自动刷新
   - 启用后列表会每 2 秒自动更新一次
   - 绿色圆点闪烁表示自动刷新已开启
   - 适用于多设备同步场景（Go 版本）

## API 接口

- `GET /` - 返回 HTML 页面
- `GET /api/items` - 获取所有剪贴板项目（置顶项在前）
- `POST /api/add` - 添加新的剪贴板项目
- `POST /api/delete` - 删除指定项目（需要提供 id）
- `POST /api/toggle-pin` - 切换项目的置顶状态（需要提供 id）

## 注意事项

- 浏览器需要支持 Clipboard API（现代浏览器都支持）
- 首次访问时浏览器可能会请求剪贴板权限，请允许
- 数据存储在内存中，服务器重启后会丢失
